Knowledge Test

There are two types of Knowledge Tests:

  • Practice Test
  • Final Test

And they can include the following types of questions:

  • Matching
  • Multiple Response
  • Multiple Choice
  • Short Answer

Practice Test

Introduction

The practice test starts out with an introductory section that includes instructions and a legend.

Each question of the Practice test has three stages:

  • Question
  • Your Results
  • Correct Answer

Question Type: Matching

Question Type: Multiple Response

Question Type: Multiple Choice

Note: All question types can have images attached for reference.

Question Type: Short Answer

 

Summary

Once completed, the user will receive a Results Summary that that includes the number of right answers in (x/y) format and the percentage scored. The summary also includes a detailed view of the user's selections.

Final Test

There is one main difference in functionality between a Practice test and a Final Test:

  • The Final Test not provide the ability to review one's answer selection after each question. The user will only receive a Results Summary upon completion of the test.

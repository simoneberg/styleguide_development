Assessment Wizard

The assessment wizard allows the user to quickly assess their skills in the Compass application. This wizard can be launched from the dashboard, Competency Inventory and can also be launched from outside of compass.

Layout

The layout for the Assessment Wizard is slightly different than the global Compass application. The header links are reduced to:

  • The Person's Avatar (Profile picture) and Name
  • Compass (a link that redirects the user back to the application)
  • Help
  • Log Out

View Demo

Other Features:

The other features that are added or removed for the Assessment Wizard are:

  • The main navigation is removed.
  • The pagination and "Next/Previous" buttons are added.

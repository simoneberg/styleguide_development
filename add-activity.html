Add Activity

When a user clicks on the "Add Activity" button from their Development Plan page, they will be redirected to a page (shown below) that lets the user add an existing activity, create a new ad-hoc activity or historic activity to their plan.

Note: Previously, the "Add Activity" area would show up in a modal/pop-up window. It is now being changed to a page redirect because this area shows more data than before and would not fit in a modal/pop-up window.

Existing Activity

When the "Add Activity" page is loaded, the "Existing Activity" option is chosen as a default. The user has to modify the following settings:

  • Search Activities: User can search for the activity by name and/or code. This feature has a auto-completion filter. As the user types each character, the search results will filter according to what was typed. For example, if the user types in "a", all search results with the letter "a" will display. The All option for the search let the user select classifiers and subclassifier to apply as filters to the serach results.
  • Date Information: Once the user selects their activity (shown in "Your Selection"), they must assign a planned start and planned end date for it. The end date must be after the start date. The priority is set to medium by default and can be changed by the user.

Create New Activity

When the user selects the "Create New Activity" radio button, the page will load a collection of controls that display the information shown below. The user needs to enter at least a Code, Name, Planned Start and Planned End in the activity information section in order to proceed with this step. The items listed under optional (custom properties) are not required to create a new activity. The priority is set to medium by default and can be changed by the user.

Historical Activity

When a user selects to add a historical activity they will need to enter the following information to create it.

  • Search Activities: User can search for the activity by name and/or code. This feature has a auto-completion filter. As the user types each character, the search results will filter according to what was typed. For example, if the user types in "a", all search results with the letter "a" will display. The All option for the search let the user select classifiers and subclassifier to apply as filters to the serach results.
  • Date Information: Once the user selects their activity (shown in "Your Selection"), they must assign a Completion Date for it.

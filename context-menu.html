Context Menu

Note: this is a temporary solution for the current version of Compass and not to be implemented in the new redesign.

Due to the removal of the Telerik controls, this page discusses an alternative solution to the context menu (when the user right-clicks).

Shown below are examples of context menu placements throughout the Compass UI. The context menu actions are placed under the header of the page. In the example below, the "Create Report Template" button is placed under the Activity Profile title.

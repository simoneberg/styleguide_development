Global Header & Footer

The global header sits at the top of all pages in the site. It includes the PetroSkills' product logo, links to top-level pages that allows access to back-end features such as Settings and Logout.

Header Structure

The header is divided by 2 main columns. The logo is placed on the left side and the user's avatar/company logo along with the top-level links are placed on the right side.

Language Selector

Language Selector allows the user to select their language of preference. This selector is controlled by a select dropdown. The select dropdown will expand and display available languages provided by Compass.

Chevron Global Header

As shown above, Chevron's Global Header is displayed differently and adheres to the Chevron's Design Standards. When designing for Chevron in Petroskills products, please use the above image as a reference. It must match the spacing that surrounds the Chevron logo and the white background. In addition, the user profile links are displayed on the top left corner of the header.

Chevron Header Structure

The grid on the Chevron header is the same as any PetroSkills products.

Language Selector allows the user to select their language of preference. This selector is controlled by a select dropdown. The select dropdown will expand and display available languages provided by Compass.

PetroCore Footer

The footer is located at the bottom of every page. It has a height of 40 pixels. It consists three key information:

  • Logo & Motto: Placed on the left.
  • Copyright Information: Placed on the middle.
  • Feedback Link: Placed on the right.

Chevron Footer

For Chevron, a footer was designed to contain links to key references in their corporation.

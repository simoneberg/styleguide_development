Tables

A table element is used for displaying data tables. The design consists of a striping background on every other row, dark gray header to emphasize the table headers, and an arrow to indicate the sorting/filtering of a data column.

View Demo

Creating Compass Tables

This is the minimum structure of a basic table element:


<table>
  <thead>
    <tr>
      <th>Header</th>
      <th>Header</th>
      <th>Header</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Cell</td>
      <td>Cell</td>
      <td>Cell</td>
    </tr>
  </tbody>
</table>

Table headers are required for every table and must be coded within this structure:


<thead>
  <tr>
    <th>Header</th>
    <th>Header</th>
    <th>Header</th>
  </tr>
</thead>

Table data must be contained in this structure:


<tbody>
  <tr>
    <td>Cell</td>
    <td>Cell</td>
    <td>Cell</td>
  </tr>
</tbody>

To make the table expand full width, add one of the grid numbers as a class. Example:


<table class="twelve">

CSS Code:

The table is styled with the following CSS code.


table {
  border: 1px solid #8a8a8a;
  -webkit-border-radius: 0;
  -moz-border-radius: 0;
  -ms-border-radius: 0;
  -o-border-radius: 0;
  border-radius: 0;
  border-spacing: 0;
}

table thead tr th {
  border-bottom: 1px solid #e1e1e1;
  background: white;
}

/* add table row striping, works in all browsers except IE8 browser. */
table tbody tr:nth-child(even) {
  background: #f1f1f1;
}

table tbody tr td {
  vertical-align: middle;
}

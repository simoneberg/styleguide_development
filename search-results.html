Search Results

This page documents how the Search Results work in the Compass application. It covers the experience of a user starting from a textual search to the page that displays the search results.

Search Workflow

The search workflow is a 2-step process. The user does:

  1. a search by typing in what they're looking for in the Search box.
  2. the user gets their search results on the main content side of the page.

Simple Search

This is where the user types in to do a simple search.

Advanced Search Feature

This control activates the feature to do an advanced search. The menu will slide down like an "accordion".

After an advanced search is made, the search results counter will display differently than a simple search. The Advanced Search results counter will contain 2 key pieces of information that relate with the Advanced Search feature along with the number of results found:

  • How many filters applied
  • What search type is applied

The results will contain 2 links that users can click on: (1) Filters Applied and (2) Search Type. When one of these links are clicked, it will flash or glow the area that corresponds with the appropriate section. Please refer to the screenshot below.

These 2 links will also cause the Advanced Search menu to open if it is closed.

  • Last Name
  • First Name
  • Employee Number
  • User ID

  • Viewable - this column will only display if the user has the permission to search people that are not assigned to them.
